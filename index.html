Course Outline

Over the period of three months, you will learn:

  • Data Science Fundamentals

    Learn the Foundations: Tools, Mathematics and Best Practices

  • Exploratory Data Analysis

    Learn how to collect and begin the exploratory process. The CRISP-D model and data visualisation

  • Statistical Modelling

    Understand the probabilistic model behind data science.

  • Machine Learning Models and Time Series Data

    Learn a variety of machine learning models and how to apply them in data science from supervised learning to unsupervised learning methods.

  • Natural Language Processing (NLP)

    Understand how NLP works and is able to create value for a data driven organisation.

  • Big Data

    Learn Hadoop and Spark. How to apply big data technologies for your data pipelines.

  • Advanced Topics and Career Specific Topics

    Choose what you want to be taught from the latest trends and advanced techniques such as deep learning and computer vision.
